# Archived Docs

This directory contains historical documentation that is kept for reference but should not be treated as current project guidance.

## What Lives Here

- Session handovers
- Dated improvement plans
- Generated critiques and review writeups
- Superseded deployment and testing guides
- Old planning documents and design explorations
- Temporary snapshots that were retained for history

## Usage

Use archived documents for background only.

If a document here conflicts with `README.md`, `AGENTS.md`, `ROADMAP.md`, or [`docs/README.md`](../README.md), treat the archived document as stale.
